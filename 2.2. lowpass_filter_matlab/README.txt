README for lowpass.m
Author: Nikita Konstantinov
Date: March, 2024.

Алгоритм реализует фильтр нижних частот для трёх сигналов: синусоида, пилообразный сигнал, меандр.

Для каждого сигнала отображается окно с шестью графиками: оригинал сигнала, сигнал+шум, выход фильтра, а 
также спектры для каждого из них.

Оригинальный сигнал зашумляется Аддитивным Белым Гауссовским Шумом (АБГШ). 
Спектр рассчитывается с использованием быстрого преобразования Фурье.
