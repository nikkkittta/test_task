README for find_rdm.m
Author: Nikita Konstantinov
Date: March, 2024.

Алгоритм создан для решения задачи поиска источника радиоизлучения (ИРИ) используя разностно-дальномерный метод.

Разностно-дальномерный метод предполагает построение гипербол и нахождение точки их пересечения. 
Для построения гипербол необходимо рассчитать значение dl - разница расстояний от источника до приёмников. Вычисление dl производится согласно формуле: dl(ab) = dt(ab) * C, где dt(ab) - разность хода между маяками a и b, C- скокрость распространения звука
Гиперболы были построены согласно формуле: sqrt((x-Xa) + (y - Ya)) - sqrt((x-Xb) + (y - Yb)) = dl(ab).

Опираясь на входные данные:
Kx = [5000; 1000; 9000];
Ky = [6000; 1000; 1000];
dt12 = 1.47 * 1e-6;
dt13 = -13.4 * 1e-6;


В результате выполнения программы координаты ИРИ: (2000, 4000).
